# Esse é meu primeiro projeto que enviei para o Github.

## Como enviar projetos para o Github:
- Precisa ter o **GIT** instalado e o **GitHub Desktop**(para facilitar e enviar os arquivos em uma tela e não por terminal)
- Abrir o **CMD** na pasta do projeto e executar o comando: `git init`
- Fzzer o **commit** (colocar o nome obrigatoriamente e descricão caso quiser) no GitHub Desktop e depois clicar em *push* para enviar
    - Se é a primeira vez, clicar em `publish`
    - Lembrar de preferencialmente, deixar o projeto como **public**

## Como baixar projetos:    
Abrir o **CMD** na pasta desejada para clonar e digitar `git clone [link do repositório]`
